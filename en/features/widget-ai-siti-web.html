Admin features

What an admin can do with an online visitor.

From the Bortex dashboard, the admin does not wait for an end-of-day report. They work with the visitor while the visitor is still on the site.

01

See live visitors

The table shows visitor, token, domain or current page, country, city, online status and time since the last activity.

02

Open the visitor chat

Clicking the row opens a conversation window with history, message input and real-time updates.

03

Manage multiple chats at once

Windows can stay open, be brought forward, minimized, restored or closed.

04

See when the visitor is typing

The typing indicator shows when the customer is replying, with live updates in the admin chat.

05

Invite to a call

The admin can generate a call link, send it to the visitor, open the room and copy the join link.

06

Enable calls from the widget

The visitor can see the "Call operator" option only when the admin enables it for that conversation.

07

Answer incoming calls

When the visitor calls, the admin receives a popup with ringtone and can answer, reject or let the request expire.

08

Save the lead

From the chat menu, the admin can save the visitor's name, email and notes to find them again among saved users.

09

Receive notifications

New messages, missed calls, call rooms and offline communications enter the dashboard notification system.

Installation and setup

One widget for every website, domain or store.

From the Widget page you can create multiple widgets, assign a name, connect the authorized domain, copy the snippet and preview desktop or mobile before publishing.

Create the widget

Set name, domain and, when available, the automatic Shopify connection.

Copy the snippet

The code uses `loader.js` with API key and widget ID, so the site loads the correct configuration.

Customize experience and AI

Configure style, messages, FAQs, notifications, quick buttons and mobile behavior.

Measure and intervene

See live visitors, tracked events, conversions and conversations directly in the dashboard.

Widget and conversion

Do not wait for the visitor to look for contact.

The widget can invite, guide and reply even before human intervention. The features are designed to reduce abandonment and increase qualified contacts.

AI

Automated replies

The assistant replies in chat with the name, tone, colors and messages configured by the brand.

FAQ

Clickable FAQs

The admin creates predefined questions and answers: the visitor opens them as quick buttons in the chat.

IN

Invitation messages

A bubble next to the widget can appear after a configured delay to encourage opening the chat.

WB

Welcome and follow-up

The chat can open with an initial message, a custom sender and quick questions.

P

Web push

The widget can request notification permission and the admin can receive alerts about new visitors or messages.

M

Mobile fullscreen

On smartphones, the conversation can fill the screen with optimized input and actions.

Tracking e analytics

Understand what happens before the request.

The widget also loads remote tracking. The dashboard uses these events to show counters, breakdowns and details by domain, tracker, pages and countries.

Shopify and e-commerce

The widget can become a salesperson for the store.

For Shopify, Bortex recognizes the store, connects the widget to the domain, enables e-commerce tracking and can show contextual AI replies. When the customer has a question about a product, shipping or order, the chat becomes an assisted sales point.

S1

Shopify widget creation

The Widget page detects Shopify installations and can automatically fill domain and source.

S2

Connected tracking

The Shopify connection can activate store tracking and show dedicated badges in the dashboard.

S3

Replies and cards

The chat runtime supports product cards, service cards and availability when the AI returns structured data.
